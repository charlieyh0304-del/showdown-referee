# 쇼다운 심판 앱 - 웹 버전

시각장애인 스포츠 쇼다운 경기를 위한 심판용 웹 앱입니다.

## 🌐 사용 방법

### 1. GitHub Pages로 배포하기

1. GitHub 저장소 생성
2. `index.html` 파일 업로드
3. Settings → Pages → Branch: main 선택 → Save
4. 생성된 주소로 접속!

예: `https://사용자명.github.io/저장소명/`

### 2. 로컬에서 테스트

`index.html` 파일을 브라우저로 열기만 하면 됩니다!

## ✨ 주요 기능

- ✅ **완벽한 접근성 지원**
  - 음성 안내 (TTS)
  - 스크린 리더 호환
  - 고대비 모드
  - 큰 터치 버튼

- 🎯 **경기 기록**
  - 개인전 (3세트/5세트)
  - 팀전 (31점)
  - 실시간 점수 입력
  - 골 득점 (+2점) / 상대 실수 (+1점)
  - 서브 카운터 자동 관리

- 💾 **자동 저장**
  - 브라우저 로컬 스토리지 사용
  - 새로고침해도 데이터 유지

## 📱 모바일에서 사용

**홈 화면에 추가:**
- Chrome: 메뉴 → "홈 화면에 추가"
- Safari: 공유 → "홈 화면에 추가"

마치 진짜 앱처럼 사용 가능!

## 🔧 기술 스택

- 순수 HTML/CSS/JavaScript
- Web Speech API (음성 안내)
- localStorage (데이터 저장)
- 의존성 없음!
